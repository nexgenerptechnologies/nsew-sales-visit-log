# Nsew Sales Visit Log

Sales Visit Log Custom App

## Features
- Client script for Sales Visit Log DocType
- Auto calculates distance and amount
- Auto fetches "Created By" name

## Installation

You can install this app using the bench command:

```bash
bench get-app https://github.com/nexgenerptechnologies/nsew-sales-visit-log.git
bench --site [your-site-name] install-app nsew_sales_visit_log
```

## Note
Ensure you export the `Sales Visit Log` DocType into this custom app from your Frappe site if it isn't already created natively.
